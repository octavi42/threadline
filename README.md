# Threadline

**Threadline** is a macOS app that shows live context from your AI coding sessions (Claude Code, Codex CLI, and Cursor).

Install and use the overlay — see **[overlay/README.md](overlay/README.md)** for full setup, hotkeys, and configuration.

```bash
cd overlay && ./install.sh
threadline-overlay show    # open the window
threadline-overlay toggle  # global hotkey: Control+Option+Command+T
```

Product design notes live in [`docs/`](docs/).
